## Summary
In this lab, we played a little bit with a memory system implementation using a latch. We were able to save 8 bit numbers to the board using one of the buttons to trigger the save. Just like in the lab instructions, it used the example of Github where it waits for the user to fully input their password to log in by triggering the enter button. This lab had a similar idea. We set the button C to be used as our trigger. Once we set the 8-bit number of our choice, we pressed the button C to save each input to one of the memory locations. Then, we are able to flip through each memory location and the number saved into each location should match the number we previously saved, until we overwrite it with another number.
